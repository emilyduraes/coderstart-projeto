Visual Studio Code

Desenvolvido pela Microsoft em 2015, é um editor de código fonte para Windows, Linux e MacOS. Para iniciantes é muito indicado por ajudar com apontamento de erros, customização de temas e realce de tags.

Suporta inúmeras linguagens de programação, como C++, Java, PowerShell, Python, Ruby, Rust, SQL, entre outras, fornecendo juntamente complementação inteligente de código como CSS, HTML, JavaScript, JSON, Less, Sass, TypeScrip. É possivel também ser extendido por plugins disponíveis através do repositório central. Por incluir o suporte git facilita na hora de guardar seus projetos em repositórios.

De uma forma mais prática e direta:

“Em sua essência, o Visual Studio Code possui um editor de código fonte super rápido, perfeito para o uso no dia-a-dia. Com suporte para centenas de idiomas, o VS Code ajuda você a ser instantaneamente produtivo com realce de sintaxe, correspondência de colchetes, recuo automático, seleção de caixa, snippets e muito mais. Atalhos de teclado intuitivos, fácil personalização e mapeamentos de atalhos de teclado contribuídos pela comunidade permitem que você navegue pelo seu código com facilidade.” - Site Oficial

HTML

O acrônimo HTML tem origem na lingua inglesa, sendo equivalente a Hypertext Markup Language, ou em português Linguagem de Marcação de Hipertexto.

É uma das linguagens usada para desenvolver web sites e facilita o entendimento tanto humano quanto da máquina, o que facilita a sua interpretação e desenvolvimento.

Funciona com o sistema de tags de marcação na qual cada uma dessas tem uma diferente função e garante se a página as exibirá ou não.

CSS

CSS é a sigla em inglês que indica o termo Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas.

Utilizado para estilizar textos, separar conteúdo e dar todas as formas desejadas aos elementos da página também podendo criar tabelas ajustar imagens e alterar todo o layout da página.

JavaScript

JavaScript, ou JS, é uma linguagem de programação de alto nível e dinâmica. Utilizado em conjunto com o HTML e o CSS, permite interatividade nas paginas Web. Foi originalmente implementada como parte dos navegadores web para que scripts pudessem ser executados do lado do cliente e interagissem com o usuário sem a necessidade deste script passar pelo servidor.

É uma linguagem multiparadigma, suportando estilos orientados a eventos, funcionais e imperativos, também incluindo a orientação à objetos.

Tipos de dados

Em seus tipos de dados têm-se Boolean (True e False), Null – nulo, Undefined – indefinido, Number – números, String – textos, Object – objetos, entre outros.

Objetos

Objetos podem ser interpretados como uma coleção de propriedades. Um conjunto de propriedades podem assumir valores de qualquer tipo, incluindo outros objetos, o que permite estruturas de dados mais complexas.

Python

Python é uma linguagem de programação de alto nível que permite fácil leitura em poucas linhas, o que a diferencia das demais linguagens. Como é uma linguagem para fácil leitura e com visual agradável a separação de blocos de código é feitas por indentação.

Por ser multiparadigma pode ser orientada a objetos, imperativo, funcional e procedural. Tem uma biblioteca padrão que contém classes, métodos e funções para realizar todas tarefas, indo de acesso a bancos de dados a interfaces gráficas com o usuário.

Contruções

As construções de Python incluem: estrutura de seleção (if, else, elif); estrutura de repetição (for, while), construção de classes (class); construção de sub-rotinas (def); construção de escopo (with).

Tipos de dados

seus tipos de dados têm-se Str (string) - que são os textos, List – lista heterogenia mutável, Tuple – tupla imutável, Int - números inteiros, Float – números quebrados, Bool – booleano (True e False), entre outros.

Django

Django é um framework para rápido desenvolvimento web escrito em Python, seu padrão é o model-template-view (MTV) e permite que o desenvolvedor reutilize a maior parte do código já feito, evitando repetições.

Padrão MTV

Model: onde acontece toda a interação com o banco de dados.

Template: renderização dos dados, ou seja, interação do usuário com o site.

iew: as regras de negócios e sistema Django.

Principais Caracteristicas

Com o Django é possível gerar tabelas no banco de dados e manipula-las através de classes em Pythom, criar interfaces administrativas, formulários, URLs amigáveis de maneira simples, sistema de separação de design, conteúdo e códigos, sistema de cache e suporte para aplicações multi-idiomas.

Por que usar Django?

Você pode se concentrar em escrever seu código enquanto o Django se encarrega do desenvolvimento da Web, cuidando da autenticação do usuário, administração de conteúdo e mapas do site, por exemplo.
